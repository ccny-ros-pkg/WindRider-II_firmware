	const std::string execute(const std::string& command)

Execute method.

This method processes a raw received command.

#. Seperate token and space seperated args from the raw command.

#. Lookup the token in the command table.

#. In case the token exists in the table, execute the corresponding commnad handler passin the space seperated argument.

#. Return a status string.



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- arg

		- Raw command to execute as received from host.

Status string.
